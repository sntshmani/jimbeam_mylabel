# bowmore-front

## Installation

### Step 0 - Environment needed:
- brew - https://brew.sh/
- nvm - https://github.com/creationix/nvm
- yarn - ```brew install yarn --ignore-dependencies```
- node v10 via nvm - ```nvm install 10```

### Installation
- Clone this repo
- set node 10 version ```nvm use 10```
- run ```yarn```
- run ```yarn dev```

### Build
- run ```yarn build```
